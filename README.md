# Execução Fiscal de IPTU — Landing Page

Página estática (`index.html`) para captação de clientes do escritório Medeiros & Costa Advocacia, voltada a pessoas que receberam cobrança ou foram citadas em execução fiscal de IPTU no Rio de Janeiro.

## Conteúdo

- Explicação em linguagem simples sobre execução fiscal (Lei 6.830/80) e formas de defesa
- Sinais de que a dívida pode ser contestada (prescrição, erro de cálculo, titularidade, isenção)
- Chamadas para contato via WhatsApp e telefone
- Perguntas frequentes

## Tecnologia

Página única em HTML/CSS puro, sem build step ou dependências. Fontes carregadas via Google Fonts (Fraunces, Public Sans, Roboto Mono).

## Como visualizar localmente

Abra `index.html` diretamente no navegador, ou sirva a pasta com qualquer servidor estático, por exemplo:

```
npx serve .
```
